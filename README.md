# Cloudysk
A Node.js based file browser.

## Install
```bash
# Clone project
git clone https://github.com/leeyee11/cloudysk.git
cd cloudysk
npm ci

# Setup frontend
git clone https://github.com/leeyee11/cloudysk-frontend.git frontend
npm ci
npm run build
cd ..

# Start running
npm run start
```

## Usage
Open `http://localhost:3000` in browser
